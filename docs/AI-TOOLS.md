## PARTE 2 — PROMPT PARA LOVABLE.DEV

Cole este prompt no [Lovable](https://lovable.dev) para gerar um website profissional:

```
Create a professional business website for a services-based company with the following specifications:

### Brand & Design
- Color Scheme: Primary blue (#1F4E79), accent red (#C00000), white (#ffffff) backgrounds
- Style: Professional, modern, clean corporate design
- Typography: Clear, readable sans-serif fonts (Segoe UI preferred)
- Layout: Fully responsive (mobile, tablet, desktop)
- Language: Portuguese (pt-BR)

### Pages Required
1. Homepage (Landing Page)
   - Hero section with compelling headline and dual CTA buttons
   - 6 service cards in a grid layout
   - About section (2-column: text + visual)
   - Stats section (4 key metrics)
   - Why Choose Us section (6 cards)
   - 3 testimonials with 5-star ratings
   - Contact/inquiry section with form
   - Professional footer with social links

2. About Page
   - Company mission, vision, and values
   - Company history/milestones
   - Team section
   - Why Choose Us section with stats

3. Services Page
   - Detailed service descriptions (6 services)
   - Pricing table with tiers (Básico, Profissional, Empresarial)
   - Call-to-action buttons

4. FAQ Page
   - 15+ FAQs in accordion style
   - Organized in 4 categories
   - Structured data markup for SEO

5. Contact Page
   - Contact form: Nome, Email, Empresa, Serviço, Mensagem
   - Company contact info
   - Business hours

### Navigation
- Sticky header with logo + nav links
- Mobile hamburger menu
- Smooth scroll to anchor links

### Key Features
- Loading screen with spinner
- Back-to-top button
- Cookie banner
- Scroll reveal animations
- Form validation
- Print-friendly styles
- Reduced motion support for accessibility

### Performance
- Zero external dependencies
- All CSS/JS inline or in separate file
- Optimized for Core Web Vitals
```

## PARTE 3 — PROMPT PARA CLAUDE/CHATGPT (Geração de Código)

Use este prompt para gerar o site diretamente via chat AI:

```
Cria uma landing page HTML completa e auto-contida (CSS e JS inline) para uma empresa B2B chamada [NOME].

REQUISITOS:
- Idioma: Português (pt-BR)
- Cores: Primary #1F4E79, Accent #C00000, Body text #7A7A7A
- Font: Segoe UI
- Zero dependências externas
- Responsivo (breakpoints em 768px e 480px)

SEÇÕES OBRIGATÓRIAS:
1. Header sticky com logo e nav
2. Hero com H1, parágrafo e 2 CTAs
3. 6 serviços em grid (ícone emoji + título + descrição)
4. About (2 colunas)
5. Stats (4 métricas)
6. Por Que Escolher (6 cards)
7. Processo (4 passos)
8. 3 Testimonials
9. CTA section
10. Newsletter signup
11. Formulário de contato + info de contato
12. Footer com social links

FEATURES:
- Loading screen com spinner
- Back-to-top button
- Mobile hamburger menu
- Smooth scroll
- Animação fadeInUp nos cards de serviços

EMPRESA: [descrever empresa aqui]
```
