## PARTE 6 — INTEGRAR FORMULÁRIO DE CONTATO

O formulário atualmente usa `alert()` (placeholder). Para funcionar em produção:

**Opção 1 — Formspree (Gratuito, sem servidor)**
```html
<form method="POST" action="https://formspree.io/f/SEU_ID">
```
1. Criar conta em formspree.io
2. Criar formulário → copiar endpoint
3. Substituir `action="#"` pelo endpoint

**Opção 2 — EmailJS**
1. Criar conta em emailjs.com
2. Adicionar SDK: `<script src="https://cdn.emailjs.com/dist/email.min.js"></script>`
3. Chamar `emailjs.send()` no handler do formulário

**Opção 3 — Backend próprio (PHP/Node.js)**
```html
<form method="POST" action="send-email.php">
```
